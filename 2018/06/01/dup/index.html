dup和dup2的理解

发表于 2018-06-01 | 分类于 kernel |

1. dup和dup2

都用于复制现有的文件描述符。

int dup(int fd);
int dup2(int fd, int fd2);

区别：
dup返回的是当前可用文件描述符中的最小值。
dup2可以使用fd2指定新描述符的值。如果fd和fd2相等，dup2返回fd2， 而不关闭它。 否则，fd2的FD_CLOEXEC（执行时关闭）文件描述符的标志被清除，这样进程在调用exec时fd2时是打开状态。
返回的新文件描述符与fd共享同一个文件表项。
